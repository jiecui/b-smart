Author: 
Jie Cui (richard.cui@utoronto.ca)
University of Toronto & University of Texas Health Science Center at Houston

To install the MPACT toolbox:
1. Copy the files to the directory of your choice, e.g. ~/bsmart/
2. Within Matlab, go the directory where you have copied the files e.g. >> cd(‘~/bsmart/’);
3. Add the directory and its subdirectories into MatLab search path.
4. Type bsmart at MatLab prompt, i.e. >> bsmart
